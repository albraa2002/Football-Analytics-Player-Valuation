⚽ Football Analytics & Player Valuation Dashboard

📌 Project Overview
This project applies advanced data analytics to sports management, focusing on football (soccer) player performance and transfer market valuation. By analyzing a simulated dataset of 40 players across two seasons, this dashboard calculates advanced metrics like Expected Goals (xG) and Return on Investment (ROI) to identify clinical finishers and market inefficiencies.

🚀 Key Sporting & Business Insights
The "Value for Money" Index: Engineered a metric to calculate the true ROI of each player (Goals + Assists per Million € of Market Value), successfully identifying 'Hidden Gems' from the academy versus overpriced new signings.
Finishing Efficiency (xG Matrix): Visualized the delta between Expected Goals (xG) and Actual Goals using a scatter matrix. This clearly separates clinical finishers (over-performing their xG) from squad underperformers.
Tactical Output: Tracked total goal contributions by position (FW, MID, DEF) to identify tactical imbalances in the squad's attacking output.
Stadium Control Room UI: Designed a premium, dark-themed standalone HTML dashboard with crimson accents, mirroring professional sports analytics software.

🛠️ Tech Stack
Language: Python 3.x
Data Manipulation: Pandas, NumPy
Data Visualization: Plotly (Scatter Matrix, Horizontal Bar, Donut Charts)
UI/UX: Standalone HTML/CSS Grid Dashboard (Sports UI)
